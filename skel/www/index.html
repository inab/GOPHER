GOPHER - Function Prediction Server Evaluation Resource

Welcome to the GOPHER server! At the moment we are finishing the end-user interface. If you would like to participate in the server assessment send an e-mail to gopher_support@cnio.es

A graphical description of the way GOPHER works is available here as an SVG graph

GOPHER is being built on top of an XCESC (XML Continuous Evaluation System Core) modular platform, so most of the work can be reused in further assessments.

GOPHER is going to use the same standardize, reusable, XML protocol for evaluation and assessments as any future bioinformatic assessment system based on XCESC.

While we are tying up the loose ends of the GOPHER server you can start preparing your server for the assessment looking at the technical documentation.
