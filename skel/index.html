Welcome to GOPHER server! Just at this moment we are finishing the end-user interface. Any people who wants their servers to participate in the continuous assessment can send an e-mail to gopher_support@cnio.es

A graphical description of the way GOPHER works is available here as an SVG graph

GOPHER is being built on top of XCESC (XML Continuous Evaluation System Core) modular platform, so the most of the work can be reused for further assessment efforts.

As GOPHER is going to use the same standardize, reusable, XML protocol for evaluation and assessments as any future bioinformatic assessment system based on XCESC.

Meanwhile we are tying the loose ends of the server you can start preparing your server for assessment looking at the technical documentation.
